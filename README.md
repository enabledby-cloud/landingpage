# enabledby.cloud Landing Page

This repository contains the static landing page for **enabledby.cloud**. It provides a quick overview, branding, and a direct path for visitors to learn more or get in touch.

🔗 Live site: https://enabledby.cloud

## Structure
```
src/
  index.html
  static/
    styles.css
    page.js
    favicon.ico
```

## Typography Update

Following refreshed design instructions, the site now uses a high-contrast font pairing:

- Space Grotesk (display / body, weights 300 & 800)
- JetBrains Mono (accent / meta elements, ultra-thin 100 and bold 800 for extremes)

This replaces the previous Inter setup to create a more distinctive visual identity. Font weights intentionally jump between light and extra-bold for stronger hierarchy.
